# react_starter


## How to use it
* Development (view the tool)
    * run
        1. `npm i`
        2. `npm run dev`
        3. browse `localhost:3001`
    * test
        1. `npm i`
        2. `npm run test`

* Local Production (build and release)
    * build and publish to npm cloud server
        1. `npm i`
        2. `npm run build`
        3. `npm run start`

* Fully Deployed Product
